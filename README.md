## Downloads & Setup
Entpacke die ZIP. Doppelklicke auf `App/index.html` oder lade den `App/`-Ordner kostenlos auf Netlify Drop (app.netlify.com/drop) hoch, um die PWA auf deinem iPhone zu installieren.
